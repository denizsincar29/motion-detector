Детектор движения

Следит за камерой и поднимает тревогу, когда движение не прекращается достаточно долго. Короткие мелькания (муха, тень, свет) игнорируются.

Текущее состояние

Не запущено

Управление

0 с

Задержка даёт время уйти из кадра, прежде чем детектор начнёт считать твоё собственное движение тревогой.

Камера

Список появится после первого запуска (браузер показывает названия камер только после разрешения на доступ). Если камера одна, выбор не показывается — выбирать не из чего.

Чувствительность и длительность

50%

Чем выше, тем меньше изменений в кадре нужно, чтобы засчитать движение. Слишком высокая чувствительность реагирует на шум камеры и лёгкие тени.

1.5 с

Тревога сработает, только если движение непрерывно длится дольше этого времени. Короткие мелькания не считаются — это и есть защита от ложных срабатываний.

Тревога

Необязательно. Можно указать ссылку на MP3/WAV, либо выбрать файл ниже — выбранный файл имеет приоритет над ссылкой. Если ничего не задано, будет использован только голосовой анонс для скринридера.

Пока запущено, играет непрерывный тон, чья высота растёт вместе с интенсивностью движения в кадре — что-то вроде воющей сирены. Не связано с длительностью выше: реагирует на любое движение сразу, для наглядной обратной связи.
